## 🚀 Projects

### 📄 Task 01 — Landing Page (`CODSOFT_01`)

A visually appealing, responsive landing page built with HTML and CSS.

**Key Features:**
- Clean, modern layout with sections (hero, features, footer)
- Responsive design using CSS Flexbox/Grid
- Attention to typography, color palette, spacing, and alignment
- Smooth, user-friendly structure

**Skills Demonstrated:** HTML structure, CSS layout, responsive design, UI/UX principles

---

### 🧑‍💼 Task 02 — Portfolio Website (`CODSOFT_02`)

A personal portfolio website to showcase projects, skills, and contact information.

**Key Features:**
- Sections for About, Skills, Projects, and Contact
- Responsive layout optimized for desktop and mobile
- Structured and clean codebase using semantic HTML
- Styled with CSS for a professional look and feel

**Skills Demonstrated:** Semantic HTML, CSS styling, responsive layouts, personal branding

---

### 🧮 Task 03 — Calculator (`CODSOFT_03`)

An interactive, browser-based calculator capable of performing basic arithmetic operations.

**Key Features:**
- Operations: Addition, Subtraction, Multiplication, Division
- Live display screen showing user input and results
- Button layout aligned using CSS Grid
- Event-driven logic using JavaScript (event listeners, conditionals, operators)

**Skills Demonstrated:** JavaScript DOM manipulation, event handling, CSS Grid, logic implementation

## 🏁 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AnitusA/CODSOFT_INTERN.git
   ```

2. **Navigate to a task folder:**
   ```bash
   cd CODSOFT_INTERN/CODSOFT_01
   ```

3. **Open in browser:**
   ```bash
   # Simply open the index.html file in any browser
   open index.html
   ```

No dependencies or build tools required — these are pure HTML/CSS/JS projects.
